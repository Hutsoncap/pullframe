# Pullframe 1.0.2

This update polishes the launch build with clearer download actions, better browser sign-in checks, and expanded editor-friendly export options.

## Improvements

- Added clearer download button labels for downloads that will mux, transcode, or convert before saving.
- Added professional transcode audio merge support for ProRes and DNxHR downloads, with an automatic include-audio setting.
- Added subtitle output choices for original source, SRT, VTT, and ASS.
- Added visible chapter marker exports in Extras, including Premiere Pro marker export when chapter data is available.
- Added YouTube sign-in checks for Safari, Zen, Helium, and other supported browsers with clearer macOS Full Disk Access guidance.
- Added support ticket mode to feedback, requiring a reply email and showing the attached diagnostics summary before submission.
- Added standard tab and settings keyboard shortcuts.

## Fixes

- Fixed update dialog release note formatting and update download metadata handling.
- Fixed playlist and tab polish, including tab titles with descenders being clipped.
- Fixed browser sign-in checks that could report readable browser cookie sources as unavailable.

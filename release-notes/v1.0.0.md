# Pullframe 1.0.0

Pullframe is now available as a signed macOS app for downloading videos, audio, subtitles, thumbnails, chapters, and metadata.

## Highlights

- Local-first YouTube downloads powered by bundled yt-dlp and ffmpeg.
- Video, audio, subtitle, thumbnail, chapter, and metadata download options.
- Multi-tab workflows for videos and playlists.
- Trial and license activation flow with privacy-conscious diagnostics and feedback.
- Signed and notarized macOS release with automatic update support.

## Privacy

Pullframe does not send source URLs, filenames, local paths, media titles, raw command lines, clipboard contents, usernames, hostnames, or raw logs in app telemetry.
